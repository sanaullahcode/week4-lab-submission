### Lab 04: Code Debugging Techniques

**Objective:**
Learning debugging methods used during software construction.

**Debugging Process:**

1. **Problem Identification:**
   - Calculator program with logical errors
   - Subtraction operation using addition
   - Division with wrong operand order
   - No division by zero check

2. **Debugging Steps:**
   - Set breakpoints at critical sections
   - Traced variable values
   - Monitored program flow
   - Identified logical inconsistencies

3. **Bugs Found and Fixed:**

| Bug | Location | Issue | Fix Applied |
|-----|----------|-------|-------------|
| Bug 1 | Subtraction | Used + instead of - | Changed to num1 - num2 |
| Bug 2 | Division | Wrong operand order | Changed to num1 / num2 |
| Bug 3 | Division by zero | No check | Added if num2 == 0 condition |
| Bug 4 | Input validation | No error handling | Added try-except block |
